# User Management API

Simple ASP.NET Core Web API for managing users.

## Features
- CRUD operations (GET, POST, PUT, DELETE)
- Data validation using Data Annotations
- Custom logging middleware
- In-memory data storage

## Endpoints
- GET /api/users
- GET /api/users/{id}
- POST /api/users
- PUT /api/users/{id}
- DELETE /api/users/{id}

## Technologies
- ASP.NET Core
- C#
